# 项目概览

这个仓库是一个面向学员的 assessment starter，用来模拟接手已有后端仓库后的第一轮排查与补全工作。

你会看到：

- 一个可阅读的 FastAPI 项目结构
- 基础的 task service API
- 一组只覆盖基础行为的测试
- 少量需要你自己定位的问题和未完成点

建议先保证项目能基本运行，再逐步处理配置、接口行为、测试和文档。

如果你不知道从哪里开始，可以先按下面顺序检查：

1. `TASK.md`
2. `README.md`
3. `app/config.py`
4. `app/main.py`
5. `app/api/tasks.py`
6. `tests/`
